## Usage

```js
> cuba "<path-to-cbo-file>"
```
```js
> cuba.exe "<path-to-cbo-file>"
```
```js
> cuba "somefile.cbo"
```

_ps:_

_'**.cb**' file is not supported_

_'**.cbo**' file is supported only for _doctype_: '**cuba_object**'_
